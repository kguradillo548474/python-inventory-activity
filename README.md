# Activity 1 - Python Inventory Systempython-inventory-activity
The program aims to create a simple inventory system using two data structures: a list for item names and a dictionary for item prices. It should let the user add new items by entering a name and price, while preventing duplicates. It must also allow updating the price of an existing item, and display an error message if the item does not exist.
